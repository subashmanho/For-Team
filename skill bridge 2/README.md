# SkillBridge

Login, sign up, and profile with Supabase (email/password auth + profile photo).

## Setup

1. Run `supabase/schema.sql` in your Supabase SQL Editor
2. Add your Supabase URL and anon key to `js/config.js`
3. Run `npx serve .` and open `http://localhost:3000`

See `supabase/README.md` for full setup steps.
